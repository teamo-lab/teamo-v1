# Teamo v1 — 核心价值定位

> Phase 3 | 创建时间: 2026-03-15 | 状态: 完成

## 价值主张（一句话）

**"你的专属云端 Claude Code，打开浏览器就能用。"**

用户选择 Teamo 而不是竞品的理由：无需安装任何软件，打开网页即可拥有一个持久运行、记住你项目上下文的专属 Claude Code 实例。

## 差异化切入点

| 维度 | Teamo v1 | 竞品短板 |
|------|----------|---------|
| **零配置** | 浏览器打开即用 | Claude Code 需要终端 + Node.js |
| **持久记忆** | 专属实例 7x24 运行，跨会话保持项目上下文 | 竞品每次对话从零开始 |
| **透明可控** | 每一步操作实时可见，CLAUDE.md 精确控制 | Devin 黑盒模型 |
| **定价清晰** | 28 credits/天，按天计费 | Devin ACU 不可预测，Cursor credit 不透明 |
| **中国可用** | 部署在美国但通过 Teamo 前端访问，无需梯子 | Claude Code CLI 需要直连 Anthropic |

## 核心用户画像

### Persona 1: 独立开发者小张
- **角色**: 全栈工程师，一个人做 SaaS 产品
- **场景**: 需要 AI 帮忙写代码、调试、部署，但不想装 IDE 插件
- **痛点**: Claude Code CLI 很强但要配置环境；Devin 太贵（$500/月）
- **期望**: 打开网页就能让 AI 帮我写代码，像跟同事聊天一样

### Persona 2: 技术团队 Leader 李总
- **角色**: 10 人团队的技术负责人
- **场景**: 团队成员水平不一，希望用 AI 提升整体产出
- **痛点**: 现有工具要每人配置；担心代码安全和成本不可控
- **期望**: 给每个成员开通账号即可使用，成本透明可控
